# 01. 시스템 소개: 디지털 회계 원장

## 1. 개요

이 시스템은 전통적인 회계 원장을 현대적인 기술로 구현한 **디지털 회계 원장 (Digital Accounting Ledger)** 입니다. 우리 서비스에서 발생하는 모든 자금의 흐름을 기록하고 관리하는 **단일 진실 공급원 (Single Source of Truth)** 역할을 수행합니다.

모든 금융 시스템의 심장부로서, 이 원장 시스템은 다음 세 가지 회계 기본 원칙을 반드시 지켜야 합니다.

- **정확성 (Accuracy):** 모든 거래는 정확하게 기록되어야 하며, 계산상의 오류가 없어야 합니다.
- **불변성 (Immutability):** 한 번 기록된 거래는 절대로 임의로 수정되거나 삭제될 수 없습니다. 변경이 필요할 경우, 수정 거래(reversing entry)를 통해 기록을 남겨야 합니다.
- **감사 가능성 (Auditability):** 모든 거래 기록은 언제든지 추적하고 검증할 수 있어야 합니다.

## 2. 핵심 도메인 모델

이러한 원칙을 구현하기 위해, 시스템은 다음과 같은 핵심 비즈니스 객체(도메인 모델)를 중심으로 설계되었습니다.

- `Account`: 자산이 저장되는 '계정'
- `Transfer`: 계정 간의 '자금 이체' 거래
- `LedgerEntry`: 거래를 복식부기 원칙에 따라 기록하는 '원장 항목' (분개)
